# 修仙 · 御空飞行 ⚔️✨

一个基于HTML5 Canvas开发的修仙主题飞行射击游戏，体验从凡人到大乘期的修仙之路！

![游戏封面](screenshot.png)

## 🎮 游戏特色

- 🧙‍♂️ **修仙者角色**：操控修仙者御空飞行，斩妖除魔
- 🎨 **境界系统**：8个修仙境界，随着境界提升，道袍颜色和灵气特效随之变化
- ⚡ **法宝系统**：4种强力法宝，包括激光、三连发、八方散射、快速连发
- 👹 **妖兽系统**：3种不同等阶的妖兽，难度逐步提升
- 💫 **华丽特效**：灵气光环、御空飞行、法术攻击等修仙特效
- 🌌 **星空背景**：动态星辰背景，营造星海飞驰的感觉

## 🎁 法宝系统

### ⚡ 天雷剑（激光）
- 发射贯穿全屏的强力激光束
- 可同时击中多个妖兽
- 持续时间：5秒

### 🔱 三才阵（三连发）
- 同时发射3发灵力弹
- 中间直射，两侧斜射
- 持续时间：6秒

### ✨ 八卦盘（八方散射）
- 一次性向8个方向发射灵力弹
- 360度全方位攻击
- 持续时间：7秒

### 💨 疾风符（快速连发）
- 大幅降低攻击冷却时间
- 超高频率射击
- 持续时间：8秒

## 🕹️ 操作方法

### 键盘操作
- **移动**：← → 方向键 或 A/D 键
- **发射灵力弹**：空格键

### 鼠标操作
- **移动**：移动鼠标控制修仙者位置
- **发射灵力弹**：点击鼠标左键

## 🎯 游戏规则

- **初始状态**：3条寿元（生命值），炼气期境界
- **获取灵石**：击败妖兽获得灵石（分数）
  - 低阶妖兽：10灵石
  - 中阶妖兽：12灵石
  - 高阶妖兽：15灵石
- **境界突破**：收集足够灵石自动提升境界
- **法宝获取**：随机掉落的法宝道具，接触即可拾取
- **难度提升**：每获得150灵石，妖兽出现频率增加
- **游戏结束**：寿元耗尽则道消身陨

## 🚀 快速开始

### 在线游玩

访问：[GitHub Pages 在线游玩](https://seayuns.github.io/xiuxian-game/)

### 本地运行

1. **克隆仓库**
```bash
git clone https://github.com/seayuns/xiuxian-game.git
cd xiuxian-game
```

2. **直接运行**
- 使用浏览器打开 `index.html` 文件即可开始游戏

3. **使用本地服务器（推荐）**
```bash
# Python 3
python -m http.server 8000

# 或使用 Node.js
npx http-server

# 或使用 PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

## 🎮 游戏技巧

1. **优先收集法宝**：法宝能大幅提升战斗效率
2. **保持移动**：不要停留在一个位置，避免被妖兽撞击
3. **合理使用激光**：天雷剑（激光）最适合清理密集妖兽
4. **关注境界提升**：境界越高，视觉效果越华丽
5. **难度管理**：前期尽快提升分数，适应后期难度
